# 🎮 Games.com

Aplicación web desarrollada como proyecto final del curso **React JS** de CoderHouse.

Se trata de un ecommerce de videojuegos construido con React, que permite navegar por diferentes productos, visualizar su información y gestionar un carrito de compras.

---

## 🚀 Tecnologías

- React
- JavaScript
- HTML5
- CSS3
- React Router DOM
- Firebase
- SweetAlert2
- Toastify

---

## ✨ Funcionalidades

- Navegación entre categorías
- Detalle de productos
- Carrito de compras
- Confirmación de compra
- Base de datos con Firebase

## ⚙️ Instalación

Clonar el repositorio

```bash
git clone https://github.com/nicolasgentile/games.com.git
```

Ingresar al proyecto

```bash
cd games.com
```

Instalar dependencias

```bash
npm install
```

Iniciar la aplicación

```bash
npm start
```
